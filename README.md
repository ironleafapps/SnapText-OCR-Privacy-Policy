# SnapText OCR — Privacy Policy

This repository hosts the public privacy policy for **SnapText OCR**, an Android app for fast, private, on-device text recognition (OCR).

## Live Privacy Policy

📄 View it here: [Privacy Policy](https://ironleafapps.github.io/SnapText-OCR-Privacy-Policy/privacy-policy.html)

## About SnapText OCR

SnapText OCR lets you extract editable text from:
- Camera captures
- Gallery photos
- PDF documents

All text recognition happens **100% on-device** using Google ML Kit — no cloud upload, no internet required, no data collection.

## Why this repo exists

Google Play requires a publicly hosted privacy policy URL for apps that request camera and storage permissions. This repository exists solely to host that policy via GitHub Pages.
